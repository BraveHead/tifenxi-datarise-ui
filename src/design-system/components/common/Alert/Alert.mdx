<Title />

信息提示条组件。与 Callout 的区别：Alert 有边框 + 支持关闭按钮，用于页面级提示；Callout 无边框，用于内嵌上下文信息。

## 功能特性

<table>
  <thead>
    <tr><th>#</th><th>特性</th><th>说明</th></tr>
  </thead>
  <tbody>
    <tr><td>F1</td><td>四种语义变体</td><td>info / success / warning / danger，各有对应底色 + 边框色 + 图标色</td></tr>
    <tr><td>F2</td><td>ARIA role 分级</td><td>info/success 使用 role="status"（不打断读屏）；warning/danger 使用 role="alert"（打断读屏）</td></tr>
    <tr><td>F3</td><td>图标插槽</td><td>可选 icon 属性，带 aria-hidden="true"</td></tr>
    <tr><td>F4</td><td>可关闭</td><td>closable + onClose 配合使用，渲染 × 关闭按钮</td></tr>
    <tr><td>F5</td><td>closable 安全</td><td>closable=true 但无 onClose 时不渲染关闭按钮，避免死按钮</td></tr>
    <tr><td>F6</td><td>主题适配</td><td>语义色 token 自动切换深色背景；文字使用 text-fg-body</td></tr>
    <tr><td>F7</td><td>className 透传</td><td>自定义 className 追加到根元素</td></tr>
  </tbody>
</table>
