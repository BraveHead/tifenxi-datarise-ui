<Title />

分隔线组件。支持横向 / 竖向两种方向，默认装饰性（对辅助技术隐藏）。颜色跟随 `--line` token 自动切换主题。

## 功能特性

<table>
  <thead>
    <tr><th>#</th><th>特性</th><th>说明</th></tr>
  </thead>
  <tbody>
    <tr><td>F1</td><td>横向分隔线</td><td>默认方向，w-full h-px，颜色 bg-line</td></tr>
    <tr><td>F2</td><td>竖向分隔线</td><td>direction="vertical"，h-full w-px，需父容器有明确高度</td></tr>
    <tr><td>F3</td><td>装饰模式</td><td>默认 decorative=true，role="none" + aria-hidden，辅助技术忽略</td></tr>
    <tr><td>F4</td><td>语义模式</td><td>decorative=false 时 role="separator" + aria-orientation</td></tr>
    <tr><td>F5</td><td>主题适配</td><td>使用 bg-line token，深色模式自动切换</td></tr>
    <tr><td>F6</td><td>className 透传</td><td>支持自定义类名覆盖颜色等样式</td></tr>
    <tr><td>F7</td><td>HTML 属性透传</td><td>data-*、aria-* 等原生属性支持</td></tr>
  </tbody>
</table>
