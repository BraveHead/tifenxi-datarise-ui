## 功能特性

<table>
  <thead>
    <tr><th>#</th><th>特性</th><th>说明</th></tr>
  </thead>
  <tbody>
    <tr><td>F1</td><td>四种类型</td><td>positive（成功绿）/ negative（危险红）/ warning（警示黄）/ neutral（品牌蓝）</td></tr>
    <tr><td>F2</td><td>左侧色边</td><td>4px 左边框颜色随 type 变化</td></tr>
    <tr><td>F3</td><td>图标区域</td><td>36x36 圆角图标，背景色和图标色随 type 变化</td></tr>
    <tr><td>F4</td><td>默认图标</td><td>各 type 有不同默认图标：趋势上升/下降/警告/信息</td></tr>
    <tr><td>F5</td><td>自定义图标</td><td>icon 插槽覆盖默认图标</td></tr>
    <tr><td>F6</td><td>footer 插槽</td><td>底部区域，常放置 Tag + 影响指标文字</td></tr>
    <tr><td>F7</td><td>className 透传</td><td>自定义 className 追加到根元素</td></tr>
    <tr><td>F8</td><td>HTML 属性透传</td><td>data-*、aria-* 等原生属性支持</td></tr>
  </tbody>
</table>
