水平堆叠比例条组件。展示占比分布，支持 hover tooltip、内嵌标签和最小宽度保底。

<Controls />

## 功能特性

<table>
  <thead>
    <tr><th>#</th><th>特性</th><th>说明</th></tr>
  </thead>
  <tbody>
    <tr><td>F1</td><td>比例计算</td><td>按 items 的 value 计算各段占比宽度</td></tr>
    <tr><td>F2</td><td>最小宽度保底</td><td>小比例（&lt;3%）保底显示 3% 宽度，确保可见</td></tr>
    <tr><td>F3</td><td>圆角模式</td><td>rounded=true 时容器使用 rounded-full</td></tr>
    <tr><td>F4</td><td>内嵌标签</td><td>showInnerLabel=true 时，宽度 &ge; 8% 的段显示标签</td></tr>
    <tr><td>F5</td><td>Hover 交互</td><td>hover 段高亮，非 hover 段降低透明度；显示 tooltip</td></tr>
    <tr><td>F6</td><td>自定义 Tooltip</td><td>renderTooltip 回调自定义 tooltip 内容</td></tr>
    <tr><td>F7</td><td>空值处理</td><td>value=0 的段不渲染；总和为 0 时组件返回 null</td></tr>
    <tr><td>F8</td><td>ARIA 标注</td><td>role="img" + aria-label 列出各段数据摘要</td></tr>
    <tr><td>F9</td><td>主题适配</td><td>tooltip 背景 bg-surface，文字 text-fg-body 自动切换</td></tr>
  </tbody>
</table>
