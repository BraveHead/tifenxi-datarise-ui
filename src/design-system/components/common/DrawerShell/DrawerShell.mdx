统一抽屉壳组件。约 15 个业务 Drawer 迁移后共用此壳。三区固定结构：Header（固定）→ Body（滚动）→ Footer（固定）。

## 功能特性

| # | 特性 | 说明 |
|---|------|------|
| F1 | 三区结构 | `DrawerHeader` + `DrawerBody` + `DrawerFooter` 组合 |
| F2 | 三档宽度 | `sm=480px` · `md=640px` · `lg=800px` |
| F3 | Header 结构 | 标题（必填）+ 副标题（可选）+ 关闭按钮（可选） |
| F4 | Body 滚动 | `flex:1 + overflow-y:auto`，内容超高时自动滚动 |
| F5 | Footer 固定 | `border-top` 分隔，按钮右对齐，最多 2 个按钮 |
| F6 | 关闭回调 | `onClose` 控制关闭按钮渲染和点击回调 |
| F7 | className 透传 | 各子组件均支持 className 扩展 |
| F8 | HTML 属性透传 | `data-*`、`aria-*` 等原生属性支持 |
