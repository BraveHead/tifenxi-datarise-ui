<Title />

涨跌徽章组件。统一全项目涨跌显示逻辑，替换散落在多个组件中的重复实现。

**业务约定（医疗场景）**：默认涨=橙红（问题增多=警示），跌=绿。`inverse=true` 反转语义（满意度等正向指标）。

## 功能特性

<table>
  <thead>
    <tr><th>#</th><th>特性</th><th>说明</th></tr>
  </thead>
  <tbody>
    <tr><td>F1</td><td>方向自动推断</td><td>value &gt; 0 涨 / &lt; 0 跌 / = 0 持平，显示对应箭头图标</td></tr>
    <tr><td>F2</td><td>绝对值显示</td><td>始终显示绝对值，负号不出现在文案中</td></tr>
    <tr><td>F3</td><td>语义色逻辑</td><td>默认：涨=danger、跌=success；inverse 翻转</td></tr>
    <tr><td>F4</td><td>标签前缀</td><td>label 前缀（"同比""环比"），无则不显示</td></tr>
    <tr><td>F5</td><td>自定义单位</td><td>默认 %，可通过 unit 改为 pp、分 等</td></tr>
    <tr><td>F6</td><td>两档尺寸</td><td>sm=11px（KPI 卡内） / md=12px（独立展示）</td></tr>
    <tr><td>F7</td><td>className 透传</td><td>自定义 className 追加到根元素</td></tr>
    <tr><td>F8</td><td>HTML 属性透传</td><td>data-*、aria-* 等原生属性支持</td></tr>
  </tbody>
</table>
