## 功能特性

<table>
  <thead>
    <tr><th>#</th><th>特性</th><th>说明</th></tr>
  </thead>
  <tbody>
    <tr><td>F1</td><td>水平排列</td><td>默认方向，label 和 value 水平排列，baseline 对齐</td></tr>
    <tr><td>F2</td><td>垂直排列</td><td>direction="vertical"，label 在上 value 在下</td></tr>

    <tr><td>F4</td><td>语义色</td><td>label 使用 text-fg-secondary，value 使用 text-fg + font-medium</td></tr>
    <tr><td>F5</td><td>主题适配</td><td>使用 fg / fg-secondary token，深色模式自动切换</td></tr>
    <tr><td>F6</td><td>className 透传</td><td>自定义 className 追加到根元素</td></tr>
    <tr><td>F7</td><td>HTML 属性透传</td><td>data-*、aria-* 等原生属性支持</td></tr>
  </tbody>
