14 · Theme
Dark Mode 对照验证
切换顶部 Theme 开关，本页所有组件实时切换主题。下表列出重点差异 token 与特殊处理规则。
场景风险处理规则
TopNavBar 背景高hardcode slate-900 → 改为 var(--sidebar)，dark=#262626
热力图矩阵色高增加 .dark 下专用色阶变量：浅色用 oklch 调亮，避免低对比度
--primary 色调中L:#1F61E8 → D:#4F8CFF，自动调亮保证白背景可读性
--chart-2/3 涨跌色中已分别调至 #FF8648 / #34D399，对比度达标；DeltaBadge 统一封装后一处修改全局生效
Recharts tooltip中Chart wrapper 注入 contentStyle={{ background: 'var(--popover)', border: '1px solid var(--border)' }}
Skeleton shimmer中dark 下 from=neutral-700 to=neutral-600，避免浅色闪动
--elevation-sm 阴影低透明度 .10 → .30，只要组件用 var(--elevation-sm) 即自动适配
所有其他组件低使用 CSS 变量即可，无需额外处理
