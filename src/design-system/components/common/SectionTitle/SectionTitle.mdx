编号区块标题组件。对应 OcSectionTitle 的通用版。支持编号徽章、描述文字、附加内容插槽和右侧操作区。

## 功能特性

<table>
  <thead>
    <tr><th>#</th><th>特性</th><th>说明</th></tr>
  </thead>
  <tbody>
    <tr><td>F1</td><td>编号徽章</td><td>number 为数字时渲染圆形品牌色徽章</td></tr>
    <tr><td>F2</td><td>徽章自定义色</td><td>numberColor 支持 Tailwind class 覆盖默认 bg-brand-500</td></tr>
    <tr><td>F3</td><td>描述文字</td><td>description 渲染在标题下方，有编号时自动缩进 32px</td></tr>
    <tr><td>F4</td><td>extra 插槽</td><td>标题右侧附加内容（如 Badge / Tag）</td></tr>
    <tr><td>F5</td><td>action 插槽</td><td>最右侧操作区（如按钮），自动 ml-auto 推到右侧</td></tr>
    <tr><td>F6</td><td>无编号模式</td><td>省略 number 时不渲染徽章，描述不缩进</td></tr>
    <tr><td>F7</td><td>className 透传</td><td>自定义 className 追加到根元素</td></tr>
    <tr><td>F8</td><td>HTML 属性透传</td><td>data-*、aria-* 等原生属性支持</td></tr>
  </tbody>
</table>
