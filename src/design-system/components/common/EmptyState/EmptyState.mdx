空态占位组件。覆盖三种场景：无数据、筛选无结果、加载失败。支持自定义图标、文案和 CTA 按钮。

## 功能特性

<table>
  <thead>
    <tr><th>#</th><th>特性</th><th>说明</th></tr>
  </thead>
  <tbody>
    <tr><td>F1</td><td>三种变体</td><td>no-data（默认）/ no-result / error，各有默认图标和文案</td></tr>
    <tr><td>F2</td><td>默认图标</td><td>no-data =文档图标 / no-result =搜索图标 / error =警告三角</td></tr>
    <tr><td>F3</td><td>默认文案</td><td>各变体预设标题和描述，无需每次传入</td></tr>
    <tr><td>F4</td><td>自定义覆盖</td><td>title / description / icon 均可覆盖默认值</td></tr>
    <tr><td>F5</td><td>CTA 插槽</td><td>action 插槽支持任意 ReactNode（按钮、链接等）</td></tr>
    <tr><td>F6</td><td>图标背景色</td><td>各变体图标圆形背景色不同：neutral / brand / danger</td></tr>
    <tr><td>F7</td><td>className 透传</td><td>自定义 className 追加到根元素</td></tr>
    <tr><td>F8</td><td>HTML 属性透传</td><td>data-*、aria-* 等原生属性支持</td></tr>
  </tbody>
</table>
