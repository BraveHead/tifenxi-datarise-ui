## 功能特性

| # | 特性 | 说明 |
|---|------|------|
| F1 | 编号徽章 | `number` 为数字时渲染圆形品牌色徽章 |
| F2 | 徽章自定义色 | `numberColor` 支持 Tailwind class 覆盖默认 `bg-brand-500` |
| F3 | 描述文字 | `description` 渲染在标题下方，有编号时自动缩进 32px |
| F4 | extra 插槽 | 标题右侧附加内容（如 Badge / Tag） |
| F5 | action 插槽 | 最右侧操作区（如按钮），自动 `ml-auto` 推到右侧 |
| F6 | 无编号模式 | 省略 `number` 时不渲染徽章，描述不缩进 |
| F7 | className 透传 | 自定义 className 追加到根元素 |
| F8 | HTML 属性透传 | `data-*`、`aria-*` 等原生属性支持 |
