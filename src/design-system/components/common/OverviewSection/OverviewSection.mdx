## 功能特性

<table>
  <thead>
    <tr><th>#</th><th>特性</th><th>说明</th></tr>
  </thead>
  <tbody>
    <tr><td>F1</td><td>标题</td><td>必填，渲染为 h2，text-fs-18 font-semibold text-fg</td></tr>
    <tr><td>F2</td><td>描述文案</td><td>可选，渲染为 p，text-fs-14 text-fg-secondary</td></tr>
    <tr><td>F3</td><td>右侧附加</td><td>可选 extra 插槽，右对齐，text-fs-12 text-fg-tertiary</td></tr>
    <tr><td>F4</td><td>内容区域</td><td>children 渲染在标题区下方</td></tr>
    <tr><td>F5</td><td>语义 HTML</td><td>section + aria-labelledby + h2 + p</td></tr>
    <tr><td>F6</td><td>主题适配</td><td>使用 text-fg / text-fg-secondary / text-fg-tertiary token</td></tr>
    <tr><td>F7</td><td>className / 属性透传</td><td>自定义 className 和 HTML 属性挂载到 section</td></tr>
  </tbody>
</table>
